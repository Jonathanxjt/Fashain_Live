# Fashain\_

A sustainability-based e-commerce fashion platform developed in Vue3 and Vite.

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

If this gives an error on load:

```sh
npm install vue-google-maps-ui --save
```
